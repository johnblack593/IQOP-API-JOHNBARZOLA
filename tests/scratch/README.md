# scratch/

Directorio de trabajo temporal para scripts de exploración,
capturas de WebSocket y herramientas de diagnóstico.

**⚠️ NADA en esta carpeta es código de producción.**
Los archivos aquí son artefactos de ingeniería inversa,
capturas de sesión y utilidades de investigación one-off.

## Estructura

- `*.py`          — Scripts de exploración y diagnóstico
- `*.json`        — Capturas de datos WebSocket / HTTP
- `logs/`         — Salidas de diagnóstico en texto plano
